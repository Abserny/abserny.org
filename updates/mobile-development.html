Mobile Development Started

Development of the Abserny mobile application has officially begun. The mobile version will bring accessible object detection to smartphones, making the system more portable and convenient.

Why Mobile?

Smartphones offer several advantages for Abserny users:

  • Always with you - no need to carry additional devices
  • High-quality cameras built-in
  • Excellent accessibility features from iOS and Android
  • Haptic feedback for additional user feedback

See the Mobile App documentation for more details.
