Purpose

Abserny is a graduation project focused on creating accessible technology for visually impaired Arabic-speaking users. This documentation provides all necessary information for understanding, installing, using, and contributing to the system.

The project emphasizes:

  • Complete offline operation for privacy and reliability
  • Arabic language support throughout the system
  • Voice-activated controls for hands-free operation
  • Accessibility-first design principles
